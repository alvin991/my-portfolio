Password Store App

A modern, secure, and user-friendly password management web application built with Next.js, React, and Prisma. This app allows users to store, manage, and export their passwords securely, with a clean and intuitive interface.

Tech Stack

React
Next.js
TypeScript
Tailwind CSS
PostgreSQL
Docker

Features

  • Password Management: Add, edit, delete, and view stored passwords.
  • Search & Filter: Quickly find passwords using search and filter tools (by company, user ID, or registration email).
  • Export: Export your passwords securely as CSV.
  • Confirmation Modals: Prevent accidental deletions or changes with confirmation dialogs and dirty-checking.
  • Responsive UI: Clean, modern, and responsive design for all devices.
  • Owner Management: Owner selection via dropdown sourced from a centralized config.
  • Expired Passwords: Toggle to show/hide expired passwords, with visual distinction for expired entries.
  • Company Links: Company field is clickable and opens in a new tab.
  • Secure Backend: Uses Prisma ORM and PostgreSQL (or SQLite) for secure data storage.
  • Dockerized: Easily run the app locally or in production using Docker and Docker Compose.

Challenges

  • Secure backend with Prisma ORM and PostgreSQL/SQLite (secure storage, some SQL injection protection).
  • Confirmation modals and dirty-checking (prevents accidental deletions/changes).
  • Responsive UI (responsive design for all devices).
  • Owner management and user assignment (owner selection via dropdown).
  • Dockerized deployment (smooth Docker deployment and environment config).
  • Export as CSV (feature present, but security of export depends on implementation).
