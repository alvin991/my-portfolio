Home Expense App

A modern, secure, and user-friendly password management web application built with Next.js, React, and Prisma. This app allows users to store, manage, and export their passwords securely, with a clean and intuitive interface.

Tech Stack

React
Node.js
TypeScript
Bootstrap
PostgreSQL
Docker

Features

  • User Authentication: Secure login and registration with protected routes.
  • Expense Tracking: Add, edit, and delete transactions with support for categories and payment methods.
  • Export: Export your expenses securely as CSV.
  • Recurring Transactions: Manage recurring expenses and templates for automated tracking.
  • Credit Card Management: Track credit card expenses and payments.
  • Category Management: Create, edit, and organize expense categories.
  • Analytics & Charts: Visualize spending with pie charts and grouped tables.
  • Search & Filter: Powerful search and filter options for transactions and templates.
  • Settings & Customization: Personalize categories, colors, and icons.
  • Responsive UI: Modern, mobile-friendly interface built with React and Vite.
  • Backend API: RESTful API for all core features, with secure data handling.
  • Cron Services: Automated backend tasks for recurring templates and data updates.
